# Spectron-dream
Font Name: Spectron Dream 
Designer: Fabiah Alam Suborni 
Release Date: December 2025 
About the Font: 
Spectron Dream is a unique, science-themed typeface where every 
character represents a specific piece of scientific equipment. This font 
was created as a self-taught project at the age of 17.
